<h1 align="center">web-utils</h1>

> 一个轻量级的 JavaScript 工具库，旨在简化常见的 Web 开发任务。它提供了一系列实用的函数和方法

## 安装

#### 你可以通过 npm 或 yarn 来安装 web-utils：

```sh
$ yarn add --dev @tendo-sect/web-utils
```
or
```sh
$ npm install --save-dev @tendo-sect/web-utils
```
